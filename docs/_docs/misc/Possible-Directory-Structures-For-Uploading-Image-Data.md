---
title: Possible Directory Structures For Uploading Image Data
permalink: /docs/possible-directory-structures-for-uploading-image-data/
---

```
deriva/images/<specimen_rid>_<order>_customtext.<extension>
deriva/images/<specimen_rid>_<order>_customtext.thumbnail.<extension>
```
for example:

```
deriva/images/X-1234_0_customtext.czi
deriva/images/X-1234_0_customtext.thumbnail.jpeg

deriva/images/X-1234_1_customtext.tiff
deriva/images/X-1234_1_customtext.thumbnail.jpeg

deriva/images/X-1234_2_customtext.jpeg
deriva/images/X-1234_2_customtext.thumbnail.jpeg
```
